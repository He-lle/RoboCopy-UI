# RoboCopy GUI

微软 `robocopy` 的图形化操作软件。把 GUI 选项翻译成 robocopy 命令行参数并执行。

## 使用方法

```bash
python robocopy_gui.py
```

或把 `robocopy_gui.py` 发给别人，确保对方 Windows 系统有 Python 即可运行。

## 功能

### 源与目标
- **源目录**：要复制的文件夹
- **目标目录**：复制到哪里
- **文件模式**：默认 `*.*`（全部文件），可改为 `*.txt`、`*.doc *.pdf`
- **复制文件夹本身**：勾选后目标路径自动追加源文件夹名（默认勾选）
  - 例：`D:\Data` → `E:\Backup\Data`（勾选）/ `E:\Backup`（不勾选）

### 复制模式
| 选项 | 说明 |
|---|---|
| 完全同步 | 目标与源完全一致，会删除目标中多余的文件 |
| 包含子目录 | 连子目录一起复制（空目录也会创建） |
| 保留所有文件信息 | 保留数据、属性、时间、权限等（默认勾选） |
| 移动 | 复制后删除源文件，相当于剪切 |

### 筛选
- 按文件名排除（支持通配符 `*.log *.tmp`）
- 按目录名排除
- 只复制较新/较旧的文件
- 按文件大小筛选（`1024` 或 `1M`）
- 按修改时间筛选（天数或 YYYYMMDD）

### 性能日志
- 重试次数和间隔
- 线程数（越大越快越占资源）
- 包间隔（>0 可减轻网络压力）
- 详细输出
- 显示预计完成时间
- 仅列出不复制（安全预览）

## 系统要求

- Windows 10 / 11（需内置 `robocopy.exe`）
- Python 3.7+
