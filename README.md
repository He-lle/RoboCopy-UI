# RoboCopy GUI

微软 `robocopy` 的图形化操作软件。把 GUI 选项翻译成 robocopy 命令行参数并执行。

纯 Python 标准库，零外部依赖。

## 使用方法

```bash
python robocopy_gui.py
```

## 功能

### 源与目标
- 源目录、目标目录、文件匹配模式（`*.*`）
- **复制文件夹本身**：勾选后目标路径自动追加源文件夹名

### 复制模式
- 完全同步（删除多余文件）、包含子目录、保留所有信息、移动
- `/ZB` 可重启+备份、`/CREATE` 仅创建空文件、`/SL` 符号链接保留

### 筛选
- 排除文件/目录（支持通配符）
- 只复制较新/较旧文件
- 按大小、修改时间、最后访问时间筛选
- 排除已更改、包含修改时间不同的文件

### 性能与日志
- 重试次数/间隔、线程数、包间隔
- 详细输出、显示预计时间、仅列出预览
- `/LFSM` 低空间模式、`/TEE` 同时输出到控制台和日志
- 显示/隐藏各类输出信息（文件、类、大小、列表等）

### 其他
- 命令预览实时同步
- 配置保存/加载（JSON 文件）
- 支持所有 robocopy 命令行参数（含 /RH 时间窗口、/A+ /A- 属性操作、/JOB 文件等）

## 系统要求

- Windows 10 / 11（需内置 `robocopy.exe`）
- Python 3.7+
